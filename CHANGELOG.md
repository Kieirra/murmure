| `1.9.0`       | 2026-05-28 | **✨ New Features**<br>- **(Major) Smart Speech Mic** — Use your phone as a wireless microphone by scanning a QR code in Murmure, no app install needed on the phone.<br>- **(Major) Native Wayland support** on Linux via CLI.<br>- **(Major) Overlay overhaul** — Configurable overlay size, real-time streaming preview with configurable text width, font size and max lines, and a color-coded visualizer per recording mode (standard, LLM, command).<br>- **Documentation** — Official Murmure documentation site (configuration, limitations, tips). https://kieirra.github.io/murmure/<br>- **Tray** — Recording indicator on the tray icon during transcription (visible even when the overlay is hidden); copy the last transcript directly from the tray menu.<br>- **Shortcuts** — Shortcut to toggle Voice Mode on/off ([#279](https://github.com/Kieirra/murmure/issues/279)); Ctrl+Shift+1-4 now directly trigger LLM transcription with the matching prompt (no separate trigger shortcut); hide the LLM Connect shortcut when the feature is disabled.<br>- **Layout** — New Extensions section in the menu; global UX/UI optimization to simplify the interface.<br>- **Rules** — Custom names for personal formatting rules.<br>- **Dictionary** — Warn when entering digits in a dictionary entry and link to the formatting rules.<br>- **Import/Export** — Include Voice Mode and Smart Mic categories.<br>- **Linux** — RPM bundle for Fedora-based distributions.<br>**🐛 Bug Fixes**<br>- **Overlay** — Error messages no longer disappear too quickly and block subsequent recordings; fix streaming auto-scroll.<br>- **macOS** — Fix auto-update (wrong artifact format) ([#301](https://github.com/Kieirra/murmure/issues/301)); Cancel shortcut (Escape) no longer blocks other apps when not recording ([#302](https://github.com/Kieirra/murmure/issues/302)); tray icon adapts to light/dark mode using a template image (thanks @fwehrling); left click on the tray icon opens the menu instead of restoring the window.<br>- **Voice Mode** — Wake word now detected when spoken mid-sentence without a pause.<br>- **Shortcuts** — Some AZERTY keys (e.g. ²) could not be bound; clarify LLM Connect vs Command shortcut descriptions.<br>- **Linux / Wayland** — Disable Smart Mic and push-to-talk on Wayland; change the default Cancel shortcut on Wayland.<br>- **Sidebar** — Disable mobile mode to prevent it disappearing on high-scaling displays; use a bundled asset for the logo to avoid subroute 404 (thanks @BorisLord).<br>- **Formatting Rules** — Preserve user formatting rules when upgrading from pre-1.8.0.<br>- **Audio** — First start/stop sound after a cold start was sometimes silent (device not warmed up).<br>- **Visualizer** — More sensitive, works well with low-gain microphones.<br>- **Logs** — Log time now displayed in the correct timezone.<br>**⚡ Performance**<br>- Shrink Rust binary by 32% (66 MB → 45 MB) and trim unused dependencies (thanks @BorisLord) ([#310](https://github.com/Kieirra/murmure/pull/310)).<br>- Skip Ollama warmup when the model is already loaded (thanks @BorisLord) ([#324](https://github.com/Kieirra/murmure/pull/324)). |

| `1.8.0`       | 2026-03-28 | **✨ New Features**<br>- **(Major) Voice Mode** — Activate Murmure by voice with customizable wake words, no keyboard needed. Includes auto-send enter, and wake word listener stays active during keyboard-triggered recording.<br>- **(Major) LLM Remote Servers** — Connect to remote OpenAI-compatible API servers for LLM Connect, with per-mode provider switching, model search, per-mode voice triggers, and onboarding flow.<br>- **(Major) Import/Export Settings** — Import and export configuration settings partially or fully. Add CLI import support (`murmure import <file>`) for sysadmin mass deployment.<br>- **Shortcuts** — Add support for mouse buttons, OEM keys (-, =, [, ], ;, ', etc.), <>, ², F13-F24. Add a shortcut in overlay to cancel recording.<br>- **Rules** — Add regex support for custom rules, improve rule labels, add "?" helper for replacement text, short text correction (auto-lowercase/remove punctuation), and rule reordering.<br>- **LLM** — Allow LLM mode reordering. Add "Voice Dictation" preset for interpreting spoken formatting commands as typographic characters.<br>- **Dictionary** — Add option to clear all dictionary entries.<br>- **About Page** — Redesigned with starry background and sponsor cards.<br>- **Autostart** — Start minimized to tray on autostart.<br>- **macOS** — Add configurable option to show/hide application in Dock.<br>- **Linux** — Add `.deb` package and `install.sh` one-liner script for Debian/Ubuntu.<br>- **Update** — Update `tauri-plugin-updater` to 2.10.0 for multi-format support (.deb, .exe, macOS Intel).<br>- **Sidebar** — Add "Release notes" link in sidebar footer.<br>- **Defaults** — Improve default settings for new users (transcription in RAM, trailing space).<br>**🐛 Bug Fixes**<br>- Fix overlay scrollbar on Windows, overlay stealing focus, overlay freeze on audio timeout.<br>- Fix double percentage display during download.<br>- Use dark mode colors consistently.<br>- Improve microphone list and keep manually selected device when disconnected. Fix crash when no microphone is available.<br>- Improve LLM accuracy with system prompts. Improve response speed of reasoning models by disabling thinking mode.<br>- Improve smart mode auto-spacing and word deduplication (parakeet fix).<br>- Fix selected text detection when clipboard already contains the same content.<br>- Fix ToggleToTalk accidental restart caused by X11 auto-repeat.<br>- Fix spurious modifier key release on macOS after paste and recording lock management.<br>- Fix wake word language-aware validation, history update, and EMA smoothing for VAD. |
